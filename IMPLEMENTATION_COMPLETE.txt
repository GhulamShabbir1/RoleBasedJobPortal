================================================================================
                           HOW IT WORKS
================================================================================

FEATURE: Admins can activate/deactivate users
DEFAULT: All users are ACTIVE by default
EFFECT:  Deactivated users cannot login

HOW TO USE (Admin):
  1. Go to User Management dashboard
  2. Find the user you want to deactivate
  3. Click the ban icon (🚫) in the Actions column
  4. Confirm the action
  5. User status changes to INACTIVE (red badge)
  6. That user can NO LONGER LOGIN

TO REACTIVATE:
  1. Find the inactive user (red INACTIVE badge)
  2. Click the check icon (✅) in the Actions column
  3. Confirm the action
  4. User status changes back to ACTIVE (green badge)
  5. User can login again

LOGIN BEHAVIOR:
  ACTIVE USER:   ✅ Can login normally
  INACTIVE USER: ❌ Cannot login - receives error message
                    "Your account has been deactivated.
                     Please contact the administrator."
